## What Was Built

`app/db/base.py` — SQLAlchemy engine and session factory using `settings.database_url` from the existing pydantic-settings config. Exposes `get_db()` as a FastAPI dependency yielder. `connect_args={"check_same_thread": False}` set for SQLite thread safety.

`app/db/models.py` — Seven ORM models using classical `Column` mapping style:
- `Department` — org unit with unique name
- `User` — email+password auth, role, optional department FK, is_active flag
- `Vendor` — supplier registry with optional contact email
- `PurchaseRequest` — core entity with title, description, category, urgency, quantity, estimated_cost, justification, status (default "draft"), requester FK, optional vendor FK, assigned_role for routing
- `ApprovalRule` — configurable amount+category thresholds with required_role and priority
- `ApprovalDecision` — per-request actor decision record with note
- `AuditLog` — immutable change log with old_status/new_status for every sensitive transition

`alembic/` — Initialized with env.py importing `Base` and all models. Online migration uses `app_engine` directly (avoids duplicate engine instantiation). Migration `3b52a6da3582_initial_schema.py` autogenerated and verified.

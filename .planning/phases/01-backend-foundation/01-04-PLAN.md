# Plan 01-04: Auth Core (Security + Dependency Injection)

## Objective

Implement JWT token creation/verification, password hashing utilities, and FastAPI dependency injection functions that extract the current user from the JWT — never from the request body. This is the security foundation all routers depend on.

## Tasks

<task id="T01" type="execute">
  <title>Create app/core/security.py — JWT and password utilities</title>
  <read_first>
    - app/core/config.py — for settings.secret_key, settings.algorithm, settings.access_token_expire_minutes
    - .planning/REQUIREMENTS.md — FR-AUTH-3 for JWT payload fields
    - .env.example — for ALGORITHM and ACCESS_TOKEN_EXPIRE_MINUTES defaults
  </read_first>
  <action>
    Create app/core/security.py with these functions:

    get_password_hash(password: str) -> str:
      Use passlib CryptContext with schemes=["bcrypt"], deprecated="auto"
      Return: ctx.hash(password)

    verify_password(plain_password: str, hashed_password: str) -> bool:
      Return: ctx.verify(plain_password, hashed_password)

    create_access_token(data: dict, expires_delta: Optional[timedelta] = None) -> str:
      Copy data dict. Set exp = utcnow + expires_delta (or settings.access_token_expire_minutes minutes).
      Use jose.jwt.encode(payload, settings.secret_key, algorithm=settings.algorithm).
      data dict must include user_id and role before calling this function (caller responsibility).
      Return encoded JWT string.

    decode_access_token(token: str) -> dict:
      Use jose.jwt.decode(token, settings.secret_key, algorithms=[settings.algorithm]).
      Let JWTError propagate — caller handles it.
      Return decoded payload dict.

    Imports: from passlib.context import CryptContext; from jose import jwt, JWTError; from datetime import datetime, timedelta, timezone; from typing import Optional; from app.core.config import settings
    Module-level: pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")
  </action>
  <acceptance_criteria>
    - app/core/security.py exists
    - get_password_hash() returns a bcrypt hash string starting with "$2b$"
    - verify_password("test", get_password_hash("test")) returns True
    - verify_password("wrong", get_password_hash("test")) returns False
    - create_access_token({"user_id": 1, "role": "requester"}) returns a string with 3 dot-separated parts (JWT format)
    - decode_access_token(create_access_token({"user_id": 1, "role": "admin"})) returns dict containing user_id=1 and role="admin"
    - py -c "from app.core.security import get_password_hash, verify_password, create_access_token, decode_access_token" exits 0
  </acceptance_criteria>
</task>

<task id="T02" type="execute">
  <title>Create app/core/deps.py — FastAPI dependency injection</title>
  <read_first>
    - app/core/security.py — for decode_access_token
    - app/schemas/user.py — for CurrentUser schema
    - app/db/base.py — for get_db and SessionLocal
    - app/db/models.py — for User model (to query DB by user_id)
    - .planning/REQUIREMENTS.md — FR-AUTH-2 (Bearer token required), FR-AUTH-4 (401 on invalid)
  </read_first>
  <action>
    Create app/core/deps.py with these dependency functions:

    get_current_user(token: str = Depends(oauth2_scheme), db: Session = Depends(get_db)) -> User:
      1. Decode token using decode_access_token(). If JWTError → raise HTTPException(401, "Could not validate credentials")
      2. Extract user_id from payload. If missing → 401.
      3. Query db.get(User, user_id). If None → 401.
      4. Return ORM User object (not a schema — caller can convert).
      Derive token from header only: oauth2_scheme = OAuth2PasswordBearer(tokenUrl="/auth/login")
      NEVER read user identity from request body.

    get_current_active_user(current_user: User = Depends(get_current_user)) -> User:
      If current_user.is_active is False → raise HTTPException(403, "Inactive user")
      Return current_user.

    require_role(*allowed_roles: str):
      Returns a dependency function. Usage: Depends(require_role("admin", "finance"))
      Inner function: if current_user.role not in allowed_roles → raise HTTPException(403, "Insufficient permissions")
      Return current_user.

    Imports: from fastapi import Depends, HTTPException, status; from fastapi.security import OAuth2PasswordBearer; from sqlalchemy.orm import Session; from jose import JWTError; from app.db.base import get_db; from app.db.models import User; from app.core.security import decode_access_token
  </action>
  <acceptance_criteria>
    - app/core/deps.py exists
    - get_current_user uses OAuth2PasswordBearer (reads Authorization: Bearer header)
    - get_current_user raises HTTPException(status_code=401) for invalid/missing/expired tokens
    - get_current_active_user raises HTTPException(status_code=403) for inactive users
    - require_role returns a callable (dependency factory pattern)
    - require_role dependency raises HTTPException(status_code=403) when role not allowed
    - py -c "from app.core.deps import get_current_user, get_current_active_user, require_role" exits 0
  </acceptance_criteria>
</task>

h = get_password_hash('secretpass')

assert verify_password('secretpass', h) is True
assert verify_password('wrongpass', h) is False

# JWT round-trip
token = create_access_token({'user_id': 42, 'role': 'finance'})
parts = token.split('.')
assert len(parts) == 3, 'Not a valid JWT'
payload = decode_access_token(token)
assert payload['user_id'] == 42
assert payload['role'] == 'finance'
assert 'exp' in payload
